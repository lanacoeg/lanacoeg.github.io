Instalasi Xubuntu 18

Posted May 12, 2019

Assalamu’alaikum, Xubuntu merupakana distro Linux yang menggunakan XFCE sebagai DE-nya, karena menggunakan XFCE distro ini merupakan salah satu distro yang ringan untuk dijalankan di laptop dengan RAM 4GB (Laptop saya wkwkwk).

Sebelum melakukan instalasi, pastikan kita telah membuat bootable-nya, untuk tutorialnya bisa dilihat di sini :

Cara membuat bootable Linux menggunakan Rufus di Windows Cara membuat bootable Linux menggunakan UNetBootin di Linux

  1. Langkah selanjutnya yaitu melakukan boot ke Flashdisk, pilih Try Ubuntu.
  2. Selanjutnya klik 2x pada icon Install Xubuntu 18.04
  3. Jendela instalasi akan muncul, pilihlah bahasa yang kalian kuasai :“v
  4. Setelah itu pilih layout keyboard (standar laptop yang ada di Indonesia yaitu layout US) centang semua checkbox :“v
  5. Lakukan pemartisian, untuk penjelasan lebih detail tentang partisi pada linux bisa kalian lihat di artikel kami yang lain : Penjelasan Singkat partisi pada linux
  6. Setelah itu pilih zona waktu
  7. Lalu isi Username dan password
  8. Tunggu hingga instalasi selesai, jika sudah selesai klik restart computer. Demikian tutorial kali ini, sampai jumpa.
