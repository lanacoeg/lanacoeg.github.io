Web Browser yang Cocok di Linux

Posted Jun 9, 2019

Assalamu’alaikum,

Web browser merupakan aplikasi wajib bagi setiap sistem operasi. Apalagi bagi kita yang sering menggunakan komputer untuk mencari informasi atau pun untuk berkomunikasi di keseharian kita.

Berikut adalah beberapa pernagkat lunak web browser yang bisa dipasang di Sistem Operasi Linux

1. Chrome by Google

Web browser besutan Google ini sangatlah populer, terlebih karena perangkat lunak ini bisa dipasang di berbagai Sistem Operasi, mulai dari Windows, Linux, MacOS, Android, iOS, bahkan Raspbery.

Untuk memasang Google Chrome di Linux, kita perlu mendownload file deb di sini

Lihat juga :

2. Firefox by Mozilla

Sebenarnya Firefox sudah terinstall secara default ketika kita menginstall sistem operasi Linux, terutama turunan Ubuntu. Karena Firefox merupakan salah satu dari aplikasi gratis dan non profit.

Untuk memasangnya kita hanya perlu ketik perintah:

sudo apt install firefox

3. Opera

Pasti kita sudah familiar dengan browser yang satu ini, mungkin banyak dari kita yang menggunakan Opera Mini pada HP Nokia yang dulu dimiliki.

Ternyata Opera juga merilis browser untuk versi desktop, termasuk Linux. Fitur yang ditawarkan pun sangat menggoda, salah satunya side pane yang terintegrasi dengan Whatsapp, Telegram, dan Messenger.

Jadi kita bisa membuka pesan tanpa perlu berganti tab, fitur yang jarang dimiliki oleh browser.

Untuk memasangnya di Linux, kita perlu mendownload file deb di [sini] ()

4. Vivaldi

Browser yang satu ini memang kurang populer, tapi fitur yang ditawarkan cukup menggoda.

Browser ini berbasis Chromium, sehingga kita bisa memasang extention dari Chrome Web Store
